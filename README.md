# CSS Navigation Menu Collection

A single-page project that showcases **30 different navigation menu styles** created using **pure HTML and CSS**.

Each menu demonstrates various UI techniques such as tabs, pill navigation, gradient effects, glass styles, and creative layouts.
This project works as a **design reference and learning resource** for building modern navigation menus without JavaScript.

---

## ✨ Features

* 🎨 30 unique navigation menu styles
* ⚡ Pure CSS (no JavaScript)
* 📱 Responsive gallery layout
* ♿ Accessible navigation structure
* 🌙 Dark mode friendly
* 🧩 Easy to copy and reuse in projects

## 🧠 How It Works

Each navigation menu uses semantic HTML:

```html id="j4c0m1"
<nav aria-label="Main Navigation">
  <ul>
    <li><a href="#" aria-current="page">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Projects</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
```

* The page displays a **gallery grid** where each cell contains a different navigation menu design.
* Each menu style has its own class:

```id="v7p3wq"

.mn-1 → .mn-30

```

Each class applies unique visual styling and hover effects.

## ♿ Accessibility

Accessibility considerations include:

* Semantic `<nav>` elements
* Keyboard-friendly links
* `:focus-visible` rings for keyboard users
* High color contrast
* `aria-current="page"` for the active navigation item
